# 🌐 Portfolio - Nelson Bandos

Bienvenue dans le dépôt de mon portfolio professionnel. Je suis un futur ingénieur spécialisé en **Réseaux & Systèmes**, passionné par la conception d'architectures sécurisées et l'automatisation des infrastructures IT.

## 🚀 Aperçu du Projet

Ce projet est un portfolio moderne, élégant et responsive, conçu pour présenter mes compétences techniques, mes services de consultation et mes réalisations académiques et personnelles.

### ✨ Fonctionnalités
- **Design Premium** : Interface soignée utilisant Tailwind CSS.
- **Mode Sombre / Clair** : Basculement dynamique du thème avec persistance (optionnel).
- **Responsive Design** Entièrement optimisé pour mobiles, tablettes et ordinateurs.
- **Sections Détaillées** : Accueil, Services, Compétences, Projets et Parcours.
- **Interactivité** : Animations fluides au survol et navigation fluide (Scroll Smooth).

## 🛠️ Technologies Utilisées

- **Structure** : [HTML5](https://developer.mozilla.org/fr/docs/Web/HTML)
- **Stylisation** : [Tailwind CSS](https://tailwindcss.com/) (via CDN)
- **Icônes** : [Font Awesome 6](https://fontawesome.com/)
- **Typographie** : [Google Fonts - Inter](https://fonts.google.com/specimen/Inter)
- **Scripting** : Vanilla [JavaScript](https://developer.mozilla.org/fr/docs/Web/JavaScript)

## 💻 Installation & Utilisation Locale

Le projet est conçu pour être extrêmement léger et ne nécessite aucune étape de compilation.

1.  **Cloner le dépôt** :
    ```bash
    git clone https://github.com/Nelson2410/nelson-bandos.git
    ```
2.  **Ouvrir le projet** :
    Il suffit d'ouvrir le fichier `index.html` dans votre navigateur Web préféré (Chrome, Firefox, Edge, etc.).
